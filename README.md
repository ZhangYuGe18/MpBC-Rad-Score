# MpBC Survival Risk Estimator

**Metaplastic Breast Cancer (MpBC) — Radiotherapy Decision Support Tool**

> ⚠️ **Research Use Only** — Not intended for clinical decision-making without qualified medical oversight.

---

## Overview

An interactive, fully static web tool for estimating overall survival (OS) probability in metaplastic breast cancer patients. Powered by a **TabPFN blend model** (TabPFN + Ridge regression) trained on the SEER MpBC cohort and externally validated on an independent Chinese multi-centre dataset.

---

## Features

- **Instant predictions** — pre-computed lookup table, no server required
- **Three time horizons** — 1-year, 3-year, 5-year overall survival
- **SHAP feature attribution** — shows which clinical factors drive each individual prediction
- **Fully static** — runs entirely in the browser, GitHub Pages compatible

---

## Input Variables

| Variable | Options |
|---|---|
| **Age** | 18–90 years (1-year precision) |
| **T Stage** | T1 / T2 / T3 / T4 |
| **N Stage** | N0 / N1 / N2 / N3 |
| **Surgical Type** | BCS · Mastectomy |
| **Neoadjuvant Response** | pCR · AT (partial/stable) · Non-CR |
| **Molecular Subtype** | HR+/HER2− · HER2+ · TNBC · Unknown |

## Model Architecture

```
Input (6 features)
    ↓
IPCW binary labels (per horizon)
    ↓ fit on training data
TabPFNClassifier  +  Ridge regression (student)
    ↓
Blend: w × TabPFN + (1−w) × Ridge
  1yr: w = 0.0  (pure Ridge)
  3yr: w = 0.1  (10% TabPFN + 90% Ridge)
  5yr: w = 0.2  (20% TabPFN + 80% Ridge)
```

**SHAP**: Ridge analytical solution `coef × (x − mean)`, scaled by each horizon's Ridge weight. Exact for the 1yr model; dominant component for 3yr and 5yr.
